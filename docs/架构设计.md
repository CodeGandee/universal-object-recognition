# 架构设计（草图）

## 模块划分
- 输入处理（`io`）：加载/校验图像输入
- 检测器接口（`detector`）：统一封装不同检测器的推理
- VLM 接口（`vlm`）：统一封装不同 VLM 的判定与打分
- 核心流程（`pipeline`）：串联检测器与 VLM，产出主目标
- 可视化（`viz`，可选）：对结果进行可视化标注

## 关键接口（示意）
- `Detector.detect(image) -> List[Region]`
- `VLM.classify(image, region) -> Label/Score`
- `Pipeline.run(image) -> MainObject`

以上仅为接口意图说明，具体签名将在实现阶段确定。
