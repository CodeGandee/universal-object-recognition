# Universal Object Recognition（通用目标识别）

一个使用“检测器 + 视觉语言模型（VLM）”组合来识别图像中主要目标的 Python 项目。

> 说明：当前仓库为项目“草图（Sketch）”，仅包含结构化设计与规划，不包含具体实现代码。

## 核心目标
- 输入：单张或多张图像文件路径
- 输出：图像中的“主要目标”类别/名称与置信提示
- 方法：
  - 先用轻量/通用目标检测器获取候选目标
  - 再用 VLM 对候选目标进行语义判定与主目标排序

## 设计原则
- 简洁：以最小可行方案串联检测器与 VLM
- 可替换：检测器与 VLM 均可按接口无感替换
- 可扩展：便于引入批量处理与多设备推理
- 可复现：提供清晰的安装与运行说明（后续补充）

## 预期工作流（草图）
1. 加载输入图像 -> 基础预处理
2. 调用“检测器接口” -> 获取候选框/区域
3. 调用“VLM 接口” -> 对候选进行语义判定与打分
4. 依据策略选出“主要目标” -> 输出结果
